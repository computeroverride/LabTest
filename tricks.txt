PYTHON BUILT-IN FUNCTIONS (EXPANDED WITH ARGUMENTS + RETURNS)
============================================================

BASIC
-----
len(x)
    → number of items in x

sum(iterable)
    → total of all numeric values

max(iterable)
max(a, b, c, ...)
    → largest value

min(iterable)
min(a, b, c, ...)
    → smallest value

sorted(iterable, reverse=False)
    → new sorted list (reverse=True for descending)

round(number, ndigits)
    → number rounded to ndigits decimal places

abs(x)
    → absolute value of x

type(x)
    → type of x

id(x)
    → unique memory identifier of object

print(value, ...)
    → print to console

input(prompt)
    → user input as string


TYPE CONVERSION
---------------
int(x)
float(x)
str(x)
bool(x)
list(x)
tuple(x)
set(x)
    → convert x into specified type

dict(iterable or pairs)
    → convert to dictionary


MATH (BUILT-IN)
---------------
pow(x, y)
    → x**y

divmod(a, b)
    → (a // b, a % b)

round(x, n)
    → rounded x to n decimals


ITERABLE HELPERS
----------------
range(stop)
range(start, stop)
range(start, stop, step)
    → range sequence

enumerate(iterable, start=0)
    → (index, item) pairs

zip(a, b, ...)
    → pair elements from each iterable

map(function, iterable)
    → apply function to each element

filter(function, iterable)
    → keep elements where function(item) is True

reversed(iterable)
    → reversed iterator

all(iterable)
    → True if all items are True

any(iterable)
    → True if at least one item is True


LIST METHODS
------------
list.append(item)
    → add item to end

list.extend(iterable)
    → add all items from iterable

list.insert(index, item)
    → insert at index

list.remove(item)
    → remove first occurrence

list.pop(index)
list.pop()
    → remove and return item

list.clear()
    → remove all items

list.sort(reverse=False)
    → sort list in-place

list.count(value)
    → number of times value appears

list.index(value)
    → first index of value

list.copy()
    → shallow copy of list


DICTIONARY FUNCTIONS
--------------------
d[key]
    → value (error if key missing)

d.get(key, default=None)
    → safe lookup

d.keys()
    → all keys

d.values()
    → all values

d.items()
    → (key, value) pairs

d.update(other_dict)
    → merge/update values

d.pop(key)
d.pop(key, default)
    → remove and return value

d.popitem()
    → remove last (key, value)

d.setdefault(key, default)
    → return value; insert default if missing

d.clear()
    → remove all items

d.copy()
    → shallow copy


STRING FUNCTIONS
----------------
str.lower()
str.upper()
    → change case

str.strip()
    → remove leading/trailing whitespace

str.split(separator)
    → split into list

str.join(iterable)
    → join items into string

str.replace(old, new)
    → replace substring

str.find(substring)
    → index of substring (-1 if not found)


FILE HANDLING
-------------
open(filename, mode)
    → return file object (read/write)

file.read()
file.readline()
file.readlines()
    → read text

file.write(text)
    → write text

file.close()
    → close file
